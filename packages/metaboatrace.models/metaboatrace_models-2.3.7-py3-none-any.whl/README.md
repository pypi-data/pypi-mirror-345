## 概要

ボートレース関連の開発で用いるモデルのパッケージ  
※ 機械学習におけるモデルではなく、クリーンアーキテクチャでのエンティティ層に相当するモジュールの集積

## インストール

```bash
$ pip install metaboatrace.models
```

## 開発環境構築

```bash
uv sync
source .venv/bin/activate
```

※ `uv` が事前にインストールされていること
